## üìã Key Information at a Glance

### What is this project?
A school-focused fantasy football league application that increases student engagement while teaching analytical and strategic thinking skills.

### Who is it for?
Schools, classes, and students who want to play fantasy football together while learning data analysis.

### What's needed to run it?
- Python 3.8+
- Django 6.0.1 (included in requirements)
- SQLite database (included)
- 2GB+ storage, 512MB+ RAM

### How do students access it?
Through a web browser at: `http://<server-ip>:8000/`

### Can I modify it?
Yes! It's built on Django, a popular Python web framework, so it's highly customizable.

### What happens next?
1. Deploy to your school
2. Create a school league
3. Invite students
4. Use fantasy football to engage and teach
5. Expand to other leagues as needed
